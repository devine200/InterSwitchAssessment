# Asset Registry Data Analysis

This project contains data analysis for the Asset Registry blockchain events.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Make sure the asset-registry-backend server is running on `http://localhost:3000`

3. Open the Jupyter notebook:
```bash
jupyter notebook asset_registry_analysis.ipynb
```

## Analysis Outputs

- `analytics.json`: JSON file containing all analytics data
- `summary.md`: Markdown summary of the analysis
- Charts are displayed in the notebook and can be saved
